
# Lampa Test

Simple API for Categories and Products.




## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DB_NAME`

`DB_USER`

`DB_PASSWORD`

`DB_HOST`

`DB_PORT`

`PORT`

`JWT_SECRET`



## Installation

Steps to run:

```bash
  npm install
  npm run db:seed
  npm start
```
